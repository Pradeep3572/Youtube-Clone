# 🎬 YouTube Clone

A responsive YouTube homepage clone built using HTML5 and CSS3.

## ✨ Features

- Responsive layout
- Fixed navigation header
- Fixed sidebar
- Video card grid
- Search bar UI
- Hover animations
- Tooltips
- Notification badge
- External YouTube video links

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Grid

## 📂 Project Structure

```
youtube-clone/
├── youtube.html
├── youtube.css
├── images/
├── README.md
├── LICENSE
└── .gitignore
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/youtube-clone.git
```

2. Open `youtube.html` (or rename it to `index.html` for GitHub Pages).

## 📱 Responsive Design

- Desktop: 4-column layout
- Tablet: 2–3-column layout
- Mobile: Responsive grid
